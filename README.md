My HTML Project

📌 About Project

This is a simple beginner-friendly HTML project. It includes basic web pages like Home, About, Contact, Gallery, and Destination pages.

🛠️ Technologies Used

HTML5
Basic file structure
Images folder (for gallery and destinations)
And basics of CSS to add colours

📁 Project Structure

project-folder/

│

├── index.html

├── about.html

├── contact.html

✨ Features
Multiple web pages connected using navigation bar
Separate pages for better organization
Image gallery section
Beginner-friendly clean structure

🚀 How to Run
1. Download or clone the project
2. Open index.html in any browser
3. Navigate through pages using menu
